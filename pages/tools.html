Departed Cloud 11 on 17 May 2022.

We introduced the use of Kanban boards using Microsoft Planner, adding one to our Teams Site for the Project Plan, the Roll Marker Project and the Assessment Task 5 Presentation. This allowed us to chunk up each project, break it into tasks and allow team members to assign themselves to tasks from the backlog and move them through stages of Up Next/In Progress/Complete. This helped a lot in distributing tasks evenly and being able to track where everything was at through the assessment period.

We were probably slightly delayed in getting started on the project as there was a lot of competing pressures from assessments and workloads in other courses and we all work full-time as well, so that made things difficult. In an ideal world we would be able to concentrate 100% on a project of this size, but that just wasn’t going to be realistic under the current circumstances.

I was really surprised to hear that Marco was leaving us, those external pressures just became too much for him which is completely understandable but I’m really grateful for all he contributed to our team over A2, A3 and A5.

We each have different skillsets and strengths and I think we took advantage of that with the tasks that we each undertook, so seeing one or two team members with more activity in GitHub is not really a huge surprise as we each took on different tasks throughout the assessment period.
