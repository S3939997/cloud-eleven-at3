Comparative Team Analysis

At Cloud 11 we have come together as a group each wanting to specialise in a different field from the next. Our group has a variety of jobs we all wish to peruse for a career in the future, a lot of which have many of the same skillsets. Jack and Jay wanting to specialise in analysing and manipulating data for employers. Kurt, Marco-Antonio, and Matthew all want to work in advanced areas of computing where you need to understand the critical digital infrastructure in their respective fields. Lastly Mason is wanting to become an app developer, where this role does need to understand computing, they do specialise in creation and design of mobile applications using their preferred programming language.

Data specialists and Senior analysts share the many of the foundational skills of digital building blocks, these typically statistical programming languages and other office applications. The core programming languages used are Python, R and SQL, these allow analysts to compare data sets at a large scale with the use of algorithms and machine learning. All this raw data is useless without creating a visualisation that is easy to read and understand for their employer, this is where the use of Microsoft 365 (Excel, PowerPoint, Word), PowerBI and Salesforce come into play.

Network Administrators are primarily built upon the digital building blocks of digital security, network administration and management, hardware management, system administration and virtualisation. Being a Network Administrator means to be an expert of both digital and physical infrastructure of their employer with the type of operating systems used at business. They tend to specialise in programming languages like Python, PERL and TCL.

Cloud Engineers and Cyber Security Analysts are built the same foundational skills of digital security, networking, and virtualisation but need to know not only how a network works but also be experts in their retrospective field. Cloud Engineers for instance will have a wide range of programming languages at their disposal such as .NET, Golong, PHP, Python and SQL. They also need to completely understand cloud computing and infrastructure, all the different cloud service providers such as AWS, Azure and GoogleGo, while improving their digital security skills to fit their role typically with blockchain, cloud and mobile operating system security. Cyber Security Analysts need to understand the foundations of both roles but have a more advanced knowledge of all operating systems along with a vast knowledge of programming languages like C, C++, Java, JavaScript, PowerShell and Python. Understanding a large variety of programming languages compliments the role further allowing them to work on a wide range of projects with any type of security asset.

App Developers while specialising in being software developers with a passion for creation of mobile applications they still need to have a complete understanding of how to use a computer much like any other IT role as applications typically developed using these programming languages Java, JavaScript, Kotlin, Swift, C#. With the advance in windows and now being able to code with Swift (used to create iOS applications) App developers are free to use whatever device has their preferred operating system. Having a complete understanding of cloud computing, networks and operating systems allows for a lot more functionality with apps as technology is evolving customers and employers are excepting more features such as complete cross-platform capabilities, cloud storage for files or game saves and an easy-to-use user interface and better user experience.

Whether you want to be a Data Specialist, Senior Analyst, Network Administrator, Cloud Engineer, Cyber Security Analyst or even an App Developer all of these require a broad range of required skills, whether that be digital building blocks, human skills or business enabling skills.
